# Lim et al. (2021) — Temporal Fusion Transformers

## Summary

Introduces TFT, a multi-horizon time-series forecasting architecture combining recurrent layers, multi-head attention, and gating mechanisms. The variable selection network (VSN) flattens all inputs Ξ_t plus a static context vector c_s through a GRN, then applies standard Softmax to produce variable importance weights v_χt. A shared value matrix W_V across all attention heads produces an averaged attention matrix Ã that is used for temporal self-explanation. Achieves quantile forecasting (P10, P50, P90); outperforms DeepAR, ConvTrans, MQRNN on benchmark datasets.

## Key claims

1. TFT's variable selection network uses standard Softmax (not sparsemax or α-entmax) for variable importance weights; this produces non-sparse weights where all variables receive some positive mass, limiting hard feature zeroing.
2. Ablation shows that removing variable selection networks increases P90 quantile loss by 4.1% on average, confirming that soft variable selection provides a measurable accuracy benefit beyond interpretability.
3. TFT's interpretable multi-head attention uses a shared value matrix across all heads and computes an averaged attention matrix Ã, enabling temporal importance visualization — but attention faithfulness is not validated against gradient-based attribution.

## Relevance to P1

TFT is the primary comparison baseline for hierarchical entmax covariate selection. Its Softmax-based VSN is the incumbent mechanism P1 aims to improve upon: replacing Softmax with α-entmax would produce sparse zero weights (hard feature selection) rather than distributed soft weights. TFT's 4.1% ablation result establishes a benchmark for how much covariate selection contributes to accuracy. The shared-value interpretable attention design is a relevant structural choice.
